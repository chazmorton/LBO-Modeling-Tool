The Tool

LBO Modeling Tool

A web-based leveraged buyout analysis platform built for learning and exploration.

📊
What it does
Takes a public company's real financial data and models what a private equity leveraged buyout would look like — projecting revenue, EBITDA, debt paydown, exit value, IRR, and MOIC over a 5-year hold period.
🔍
Company Validator
Before running any model, the tool scores a company on five LBO suitability criteria — EBITDA margin, revenue scale, existing debt load, company maturity, and free cash flow — so you know upfront if the deal is realistic.
🎲
Monte Carlo Simulation
Runs 1,000+ randomised scenarios by varying growth rate, EBITDA margin, and exit multiple to show the full distribution of possible IRR outcomes — from bear case (P10) to bull case (P90).
📡
Real Data
Pulls live financial data from Yahoo Finance via the yFinance library — revenue, margins, debt, enterprise value, and revenue growth — no API key required.
Built with
Python FastAPI NumPy Pandas yFinance Chart.js HTML / CSS / JS Claude Code
This tool is for educational purposes only and does not constitute financial advice. All outputs are model projections based on public data and simplified assumptions. Real LBO analysis requires far more depth, proprietary data, and professional judgment.
